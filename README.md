## 平台简介

BitStrat 是基于 RuoYi-Vue-Plus 框架，开发的一款支持多AI Agent 策略允许AI对加密货币交易所进行高度自定义操控的平台，支持多租户集群化部署，支持高并发大数据量执行。

## 前端运行

```bash
# 克隆项目
git clone https://github.com/tokyohost/bitstrat-ui.git

# 安装依赖
npm install --registry=https://registry.npmmirror.com

# 启动服务
npm run dev

# 构建生产环境
npm run build:prod

# 前端访问地址 http://localhost:80
```
